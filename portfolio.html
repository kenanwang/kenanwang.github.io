There are four types of machine learning projects in my portfolio:

  1. End to End projects - demonstrating the whole data science workflow including acquiring data and engineering that data to be used for analysis
  2. Implementing Algorithms - best way to intuitively understand how an algorithm works and thus how best to use and tune that algorithm, typically requires some math (linear algebra, probability, statistics, numerical computation)
  3. Processed Data - using data that has already been processed to practice a machine learning workflow generally including: explorative analysis, data clean up, feature engineering, algorithm selection, and model tuning
  4. SQL - an important skill for data analysis, learning SQL well was a priority for me
